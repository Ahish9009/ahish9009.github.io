Image Digitizer

Steps

Threshold

  • Upload the image to the thresholder below
  • Tweak the box params till the output image looks good
    • Box Size: Size of the averaging box
    • Constant: Constant to be subtracted
  • Click on download to save the image

Vectorize

  • Use the Photopea tool below to vectorize the output
  • Upload the downloaded image: File -> Open
  • Vectorize using:
    Image -> Vectorize Bitmap -> Uncheck Segmentize -> OK
  • Click on PNG/SVG to download as required
