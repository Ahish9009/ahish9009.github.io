Thresh & Vect

Steps

Thresh

  • Upload the image to the thresholder below
  • Tweak the box height/width settings till the output image looks good
  • Click on download to save the image

Vect

  • Use the Photopea tool below to vectorize the output
  • Upload the downloaded image: File -> Open
  • Vectorize using: Image -> Vectorize Bitmap -> Uncheck Segmentize -> Ok
  • Click on PNG/SVG to download as required
